Korean Reader icon pack

Contents:
- icons/apple-touch-icon.png -> Apple home screen icon (180x180)
- icons/android-chrome-192x192.png -> Android / PWA icon
- icons/android-chrome-512x512.png -> Android / PWA icon
- icons/icon-maskable-192x192.png -> Android adaptive icon
- icons/icon-maskable-512x512.png -> Android adaptive icon
- icons/icon-smooth-square-*.png -> smooth-square variants
- icons/icon-circle-*.png -> circle variants
- icons/icon-transparent-*.png -> transparent symbol variants
- icons/favicon.ico -> browser tab icon
- icons/favicon-32x32.png -> browser tab icon
- icons/favicon-16x16.png -> browser tab icon
- site.webmanifest -> manifest file
- index-head-snippet.html -> paste into your index.html <head>

Recommended usage:
1. Copy the whole icons/ folder into your project root.
2. Copy site.webmanifest into your project root.
3. Paste index-head-snippet.html contents into the <head> of index.html.
